# Strobo

内部で使ってるサーバーサイドFabricのいろいろライブラリ  
基本的に最新バージョンでしか開発したくないので最新バージョン以外はサポートする気はない

## Feature

- イベントリスナー, 多少のイベント
- BrigadierのKotlin DSL
- サーバーサイドに最適化されたアイテムの実装
- サーバーサイドに最適化されたスクリーンハンドラーの実装
- タスクスケジューラー
- ユーティリティ
